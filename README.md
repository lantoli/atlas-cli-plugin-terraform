WIP

## Installing

Install the [Atlas CLI](https://github.com/mongodb/mongodb-atlas-cli) if you don't have done it yet.

Install the plugin by running:
```bash
atlas plugin install github.com/mongodb-labs/atlas-cli-plugin-terraform
```
